# Olist E-Commerce Analytics using PostgreSQL

> A comprehensive business intelligence project that transforms transactional e-commerce data into actionable business insights through advanced SQL analytics.

---

# Executive Summary

Modern e-commerce organizations generate millions of transactional records every day, but raw data alone cannot support strategic decision-making. The true value of data lies in transforming it into meaningful insights that drive revenue growth, improve customer experience, optimize operations, and support executive decision-making.

This project presents a comprehensive analytical assessment of the Brazilian Olist E-Commerce marketplace using PostgreSQL. By applying advanced SQL techniques to real-world transactional data, the project investigates key business domains including revenue performance, customer behavior, product demand, seller effectiveness, payment trends, delivery performance, and customer satisfaction.

The analysis is structured as a progressive business case study consisting of forty carefully designed analytical problems that simulate the type of work performed by Data Analysts in modern organizations.

---

# Business Context

Olist is one of Brazil's largest e-commerce marketplaces, connecting thousands of independent sellers with customers across the country.

Every customer interaction generates valuable operational data:

- Orders
- Payments
- Products
- Sellers
- Customer Reviews
- Delivery Information
- Geographic Locations

This project demonstrates how relational databases can be leveraged to convert operational data into business intelligence that supports informed decision-making.
